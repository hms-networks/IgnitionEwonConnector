# Ignition EwonÂ® Connector Module - Allowed Tag Names

The tags synchronized by the connector must adhere to the following rules.

By default, and with the `Disable Tag Name Checking` option disabled, tags must begin with an alphanumeric or
underscore. Additional characters in tag names must be an alphanumeric, underscore, space, or any of the
following: `' - : ( )`.
